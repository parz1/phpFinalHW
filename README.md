# phpFinalHW
Final work of school class.

虽然这个RESTful有点畸形，但还是可以这样写成一套简单的api


### 接口如下
**`GET` : phpFinalHW/class**
获取所有comment

params:

1. type
2. userid

**`POST` : phpFinalHW/class**
注册/登陆/发布comment

params:

1. type 1/2/3
2. userid


**`PATCH` : phpFinalHW/class**
修改密码

params:

1. userid
2. pwd

**`DELETE` : phpFinalHW/class**
删除评论

params:

1. commentid
